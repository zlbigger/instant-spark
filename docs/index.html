💡同步到 flomo - 隐私权政策

引言

感谢您使用"💡同步到 flomo"Chrome 扩展程序。我们非常重视您的隐私权，并致力于保护您的个人信息。本隐私权政策旨在向您说明我们如何收集、使用和保护您的信息。

信息收集

我们坚持最小数据收集原则，仅收集实现核心功能所必需的信息：

  • 您的 flomo API 地址（用于同步内容）
  • 您选中的网页内容（仅在您主动触发同步时）

信息使用

收集的信息仅用于：

  • 将您选中的内容同步到您的 flomo 账户
  • 保存您的 API 设置，避免重复输入

信息存储

  • 所有数据均存储在您的本地浏览器中
  • 我们不会将任何数据上传到我们的服务器
  • 您的 API 地址通过 Chrome 的安全存储机制保存

数据传输

  • 数据传输仅发生在您的浏览器和 flomo 服务器之间
  • 使用 HTTPS 协议确保传输安全
  • 不会与任何第三方共享您的数据

权限使用

我们申请的浏览器权限及其用途：

  • activeTab：仅在您主动触发时读取选中的内容
  • storage：保存您的 API 设置
  • contextMenus：提供右键菜单快捷操作
  • notifications：显示操作结果通知
  • host_permissions：访问 flomo API 发送内容

用户控制

  • 您可以随时在扩展设置中删除保存的 API 地址
  • 您可以通过 Chrome 的扩展管理页面停用或删除本扩展
  • 删除扩展会同时删除所有本地存储的数据

隐私权政策更新

我们可能会不时更新本隐私权政策。更新后的政策将在本页面上发布，重大变更时会通过扩展程序通知您。
